Windows用画像ビューアー。

## 特徴

- 前後画像の先読みによる高速切り替え
- 多形式対応: JPEG / PNG / GIF / BMP / WebP / PDF / ZIP / cbz / RAR / cbr / 7z / 64bit Susieプラグイン
- 設定ファイルとキーバインドによるカスタマイズ

## インストール

[Releases](https://github.com/ak110/gv/releases) からZIPをダウンロードし、任意のフォルダに展開する。
