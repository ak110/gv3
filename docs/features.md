# ぐらびゅ3 機能仕様書

## 優先度の定義

- **Must**: リリースに必須。ぐらびゅ2ユーザーが移行するために不可欠
- **Should**: できれば実装。ぐらびゅ2にあった機能で有用だが、初期リリースでは省略可能
- **Could**: あれば嬉しい。新規機能や利便性向上

---

## 1. 画像表示 [Must]

### 1.1 基本表示
- ファイルを引数に起動すると画像を表示する
- Windowsのファイル関連付けから起動できる
- ドラッグ&ドロップでファイルまたはフォルダを開ける

### 1.2 表示モード
- **自動縮小表示**: ウィンドウサイズに収まるよう画像を縮小（デフォルト）
- **自動縮小・自動拡大表示**: ウィンドウサイズに合わせて拡大も行う
- **自動拡大表示**: ウィンドウサイズに合わせて拡大のみ行う
- **原寸大表示**: 100%表示
- **固定倍率表示**: 任意の倍率で表示
- 倍率の拡大(×1.4) / 縮小(×0.7) / リセット(×1.0)

### 1.3 余白
- 画像とウィンドウ端の余白量を設定可能

### 1.4 αチャネル処理
- PNG等の透過画像に対して背景色を切り替え可能（白、黒、チェッカーパターン等）

### 1.5 画像情報表示 [Must]
- 画像サイズ（幅×高さ）、ファイルサイズ、ファイルパス
- PNGのコメント等のメタデータ
- EXIF情報（JPEG）

---

## 2. ファイルナビゲーション [Must]

### 2.1 フォルダ内ナビゲーション
- 画像ファイルを開くと、同一フォルダ内の画像ファイルを自動的にリストに追加
- 矢印キーで前後の画像に移動

### 2.2 アーカイブ内ナビゲーション
- アーカイブファイル（ZIP, cbz, RAR, 7z等）を開くと、内部の画像を自動リスト化
- フォルダ内と同様にナビゲーション可能

### 2.3 移動操作
| 操作 | デフォルトキー |
|------|--------------|
| 前へ | ← |
| 次へ | → |
| 5ページ前へ | PageUp |
| 5ページ先へ | PageDown |
| 50ページ前へ | Ctrl+PageUp |
| 50ページ先へ | Ctrl+PageDown |
| 前のフォルダへ | Shift+PageUp |
| 次のフォルダへ | Shift+PageDown |
| 前のマーク画像へ | Ctrl+Shift+← |
| 次のマーク画像へ | Ctrl+Shift+→ |
| 最初へ | Ctrl+Home |
| 最後へ | Ctrl+End |
| 指定ページへ | Ctrl+Space |

### 2.4 一時的ソートナビゲーション [Should]
- ソート順で一時的に前後移動（Shift+Tab / Tab）
- 現在のリスト順序を変えずに、ソート順での隣接画像に移動

---

## 3. 先読み（プリフェッチ） [Must - 最重要]

- 現在表示中の画像の前後N枚をバックグラウンドでデコード・キャッシュ
- キャッシュ済み画像への切り替えは瞬時（描画のみ）
- 利用可能メモリに基づいてキャッシュ枚数を動的に調整
- シャッフル後もリスト順に基づいて先読みが機能する

---

## 4. ファイルリスト管理 [Must]

### 4.1 ファイルリスト表示
- ファイルリストウィンドウの表示/非表示切り替え（F4）
- リスト内のファイルをクリックして直接移動

### 4.2 ソート
- ファイル名順（デフォルト）
- 大文字小文字区別なし
- フォルダ名を除いた部分でソート
- フォルダ名を除いた部分 / 大文字小文字区別なし
- ファイルサイズ順
- 最終更新日時順
- 数値認識ソート（自然順ソート）

### 4.3 シャッフル
- ファイルリスト全体をシャッフル
- フォルダ/アーカイブの順序のみをシャッフル（フォルダ内の順序は維持）

### 4.4 ブックマーク（ファイルリスト保存/復元） [Must]
- 現在のファイルリストと表示位置を独自テキスト形式で保存
- 保存したブックマークからファイルリストと位置を復元
- ブックマークファイルをテキストエディタで開く

---

## 5. マーク機能 [Must]

### 5.1 マーク操作
- 現在のファイルにマークを設定（Delete）
- マークを解除（Ctrl+Delete）
- 全てのマークを解除
- 全てのマークを反転（Ctrl+Shift+I）
- 最初から現在までのマークを反転

### 5.2 マークしたファイルへの一括操作
- マークしたファイルをリストから削除（Ctrl+BackSpace）
- マークしたファイルを完全に削除（Ctrl+Shift+Delete）
- マークしたファイルを移動（Ctrl+Shift+M）
- マークしたファイルを複製（Ctrl+Shift+C）
- マークしたファイル名をクリップボードにコピー

### 5.3 マーク間ナビゲーション
- 前のマーク画像へ移動（Ctrl+Shift+←）
- 次のマーク画像へ移動（Ctrl+Shift+→）

---

## 6. ファイル操作 [Must]

### 6.1 単一ファイル操作
- ファイルをリストから削除（BackSpace）— ファイル自体は削除しない
- ファイルを削除（Shift+Delete）— ごみ箱経由
- ファイルを移動（Ctrl+R）
- ファイルを複製/保存（Ctrl+S）

### 6.2 クリップボード
- ファイル名をクリップボードにコピー（Ctrl+F）
- 画像をクリップボードにコピー（Ctrl+C）
- クリップボードから貼り付け（Ctrl+V）

### 6.3 画像の書き出し [Must]
- JPGファイルとして書き出す（Ctrl+J）
- BMPファイルとして書き出す（Ctrl+B）
- PNGファイルとして書き出す（Ctrl+P）

### 6.4 フォルダ操作
- 現在の画像のあるフォルダをエクスプローラで開く（Ctrl+D）

---

## 7. ウィンドウ・表示 [Must]

### 7.1 ウィンドウ操作
- 新規ウィンドウ作成（Ctrl+N）
- ウィンドウ最小化（Ctrl+X）
- ウィンドウ最大化（左ダブルクリック）
- 常に手前に表示

### 7.2 フルスクリーン
- 全画面表示切り替え（Alt+Enter）— ウィンドウ枠なし最大化
- フルスクリーン切り替え（Alt+Shift+Enter）— ディスプレイ解像度変更
- 全画面時にカーソル自動非表示

### 7.3 メニューバー
- メニューバーの表示/非表示切り替え（Esc）

---

## 8. 画像フォーマット対応 [Must]

### 8.1 標準対応
- JPEG / JPG
- PNG（αチャネル、コメント対応）
- GIF（アニメーションGIF対応は Could）
- BMP
- WebP（ぐらびゅ2からの新規追加）
- TIFF [Should]
- TGA [Should]
- ICO / CUR [Should]

### 8.2 Susieプラグイン対応 [Should]
- 64bit版Susieプラグイン (.sph / .spi) の動的ロード
- `$AppDir\spi\` ディレクトリからの自動検出
- 画像デコードプラグインとアーカイブプラグインの両方をサポート
- プラグインの優先度設定

---

## 9. アーカイブフォーマット対応 [Must]

### 9.1 標準対応
- ZIP / cbz
- RAR / cbr
- 7z

### 9.2 Susieプラグイン経由 [Should]
- Susieアーカイブプラグインで対応形式を拡張
- LHA, GCA等

### 9.3 パスワード付きアーカイブ [Should]
- パスワード入力ダイアログ

---

## 10. シェル統合 [Must]

### 10.1 ファイル関連付け
- 対応する画像形式の関連付けをレジストリに登録/解除

### 10.2 エクスプローラ連携
- 右クリックメニュー（コンテキストメニュー）への登録
- 「送る」(SendTo) への登録

---

## 11. 設定 [Must]

### 11.1 設定ファイル
- TOML形式の設定ファイル（`gv3.toml`）
- アプリケーションと同じディレクトリに保存

### 11.2 キーバインド設定
- ほぼ全ての操作にカスタムキーバインドを設定可能
- TOML形式のキーバインド設定ファイル（`gv3.keys.toml`）
- 詳細は `docs/keybindings.md` を参照

### 11.3 設定ダイアログ [Should]
- 表示設定
- 画像設定
- リスト設定
- 全般設定
- プラグイン設定
- キー設定

---

## 12. その他 [Must]

### 12.1 Unicodeファイルパス対応
- ファイルパスにUnicode文字（日本語、絵文字等）を含むファイルを正しく処理

### 12.2 画像の再読み込み（F5）
- 外部で変更された画像を再読み込み

### 12.3 履歴 [Should]
- 最近開いたファイル/フォルダの履歴（Ctrl+H）

---

## ぐらびゅ2から削除した機能

以下の機能はぐらびゅ3では実装しない：

| 削除した機能 | ぐらびゅ2での場所 |
|-------------|-----------------|
| 画像編集（左右反転、上下反転、回転、切り抜き、解像度変更） | 編集メニュー |
| フィルタ（レベル補正、ガンマ補正、ぼかし、シャープ、モザイク、色反転、グレースケール化、メディアンフィルタ、アンシャープマスク、ガウスぼかし、明るさとコントラスト） | 画像メニュー、フィルタメニュー |
| 壁紙設定 | 編集 → 壁紙に設定 |
| 外部ビューア連携 | 編集 → 外部ビューアで開く |
| 見開き表示 | 設定 → 見開き表示 |
| 拡大鏡 | 画像 → 拡大鏡を表示 |
| スライドショー（自動ページ送り） | リスト → 自動ページ送り |
| ネットワーク更新 | ヘルプ → ねとわく更新 |
| 配布サイトを開く / 作者へメール | ヘルプメニュー |
| ARGB形式での書き出し | 画像 → ARGBファイルとして書き出す |
| 描画モード切替（GDI/GPU） | Direct2D一本に統一 |
