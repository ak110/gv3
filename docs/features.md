# ぐらびゅ3 機能仕様書

## 1. 画像表示

### 1.1 基本表示
- ファイルを引数に起動すると画像を表示する
- Windowsのファイル関連付けから起動できる
- ドラッグ&ドロップでファイルまたはフォルダを開ける

### 1.2 表示モード
- **自動縮小表示**: ウィンドウサイズに収まるよう画像を縮小（デフォルト）
- **自動縮小・自動拡大表示**: ウィンドウサイズに合わせて拡大も行う
- **自動拡大表示**: ウィンドウサイズに合わせて拡大のみ行う
- **原寸大表示**: 100%表示
- 倍率の拡大(×1.4) / 縮小(×0.7) / リセット(×1.0)

### 1.3 余白
- 画像とウィンドウ端の余白量を設定可能（トグル）

### 1.4 αチャネル処理
- PNG等の透過画像に対して背景色を切り替え可能（白、黒、チェッカーパターン）

### 1.5 画像情報表示
- 画像サイズ（幅×高さ）、ファイルサイズ、ファイルパスをタイトルバーとメッセージボックスで表示

## 2. ファイルナビゲーション

### 2.1 フォルダ内ナビゲーション
- 画像ファイルを開くと、同一フォルダ内の画像ファイルを自動的にリストに追加
- 矢印キーで前後の画像に移動

### 2.2 アーカイブ内ナビゲーション
- アーカイブファイル（ZIP, cbz, RAR, cbr, 7z）を開くと、内部の画像を一時フォルダに展開してリスト化
- フォルダ内と同様にナビゲーション可能

### 2.3 移動操作
| 操作 | デフォルトキー |
|------|--------------|
| 前へ | ← |
| 次へ | → |
| 1ページ前へ | Shift+← |
| 1ページ先へ | Shift+→ |
| 5ページ前へ | PageUp |
| 5ページ先へ | PageDown |
| 50ページ前へ | Ctrl+PageUp |
| 50ページ先へ | Ctrl+PageDown |
| 前のフォルダへ | Shift+PageUp |
| 次のフォルダへ | Shift+PageDown |
| 前のマーク画像へ | Ctrl+Shift+← |
| 次のマーク画像へ | Ctrl+Shift+→ |
| 最初へ | Ctrl+Home |
| 最後へ | Ctrl+End |
| 指定ページへ | Ctrl+Space |

## 3. 先読み（プリフェッチ）

- 現在表示中の画像の前後N枚をバックグラウンドでデコード・キャッシュ
- キャッシュ済み画像への切り替えは瞬時（描画のみ）
- 利用可能メモリに基づいてキャッシュ枚数を動的に調整
- 優先度付きロード: 現在ページ > 次ページ > 前ページ > 遠いページ
- ナビゲーション時に不要なリクエストを自動キャンセル（世代管理）

---

## 4. ファイルリスト管理

### 4.1 ソート
- ファイル名順（デフォルト）
- 大文字小文字区別なし
- ファイルサイズ順
- 最終更新日時順
- 数値認識ソート（自然順ソート）

### 4.2 シャッフル
- ファイルリスト全体をシャッフル

### 4.3 ブックマーク（ファイルリスト保存/復元）
- 現在のファイルリストと表示位置を独自テキスト形式（`.gv3bm`）で保存
- 保存したブックマークからファイルリストと位置を復元
- ブックマークファイルをテキストエディタで開く

---

## 5. マーク機能

### 5.1 マーク操作
- 現在のファイルにマークを設定（Delete）
- マークを解除（Ctrl+Delete）
- 全てのマークを反転（Ctrl+Shift+I）

### 5.2 マークしたファイルへの一括操作
- マークしたファイルをリストから削除（Ctrl+BackSpace）
- マークしたファイルを完全に削除（Ctrl+Shift+Delete）— ごみ箱経由
- マークしたファイルを移動（Ctrl+Shift+M）
- マークしたファイルを複製（Ctrl+Shift+C）
- マークしたファイル名をクリップボードにコピー

### 5.3 マーク間ナビゲーション
- 前のマーク画像へ移動（Ctrl+Shift+←）
- 次のマーク画像へ移動（Ctrl+Shift+→）

---

## 6. ファイル操作

### 6.1 単一ファイル操作
- ファイルをリストから削除（BackSpace）— ファイル自体は削除しない
- ファイルを削除（Shift+Delete）— ごみ箱経由
- ファイルを移動（Ctrl+R）
- ファイルを複製/保存（Ctrl+S）

### 6.2 クリップボード
- ファイル名をクリップボードにコピー（Ctrl+F）
- 画像をクリップボードにコピー（Ctrl+C）— CF_DIB形式
- クリップボードから貼り付け（Ctrl+V）

### 6.3 画像の書き出し
- JPGファイルとして書き出す（Ctrl+J）
- BMPファイルとして書き出す（Ctrl+B）
- PNGファイルとして書き出す（Ctrl+P）

### 6.4 フォルダ操作
- 現在の画像のあるフォルダをエクスプローラで開く（Ctrl+D）

---

## 7. ウィンドウ・表示

### 7.1 ウィンドウ操作
- 新規ウィンドウ作成（Ctrl+N）
- ウィンドウ最小化（Ctrl+X）
- ウィンドウ最大化（左ダブルクリック）
- 常に手前に表示

### 7.2 フルスクリーン
- 全画面表示切り替え（Alt+Enter）— ボーダーレス最大化
- 全画面時にカーソル自動非表示（5秒タイマー）

---

## 8. 画像フォーマット対応

### 8.1 標準対応
- JPEG / JPG
- PNG（αチャネル対応）
- GIF（静止画のみ）
- BMP
- WebP

### 8.2 Susieプラグイン対応
- 64bit版Susieプラグイン (.sph / .spi) の動的ロード
- exeと同階層の `spi/` ディレクトリからの自動検出
- 画像デコードプラグインとアーカイブプラグインの両方をサポート

---

## 9. アーカイブフォーマット対応

### 9.1 標準対応
- ZIP / cbz
- RAR / cbr
- 7z

### 9.2 Susieプラグイン経由
- Susieアーカイブプラグインで対応形式を拡張可能

---

## 10. シェル統合

### 10.1 ファイル関連付け
- 対応する画像形式の関連付けをレジストリに登録/解除

### 10.2 エクスプローラ連携
- 右クリックメニュー（コンテキストメニュー）への登録
- 「送る」(SendTo) への登録

### 10.3 一括登録/解除
```bash
gv3.exe --register      # 関連付け・コンテキストメニュー・送るを一括登録
gv3.exe --unregister    # 一括解除
```

---

## 11. 設定

### 11.1 設定ファイル
- TOML形式の設定ファイル（`gv3.toml`）
- アプリケーションと同じディレクトリに保存

### 11.2 キーバインド設定
- ほぼ全ての操作にカスタムキーバインドを設定可能
- TOML形式のキーバインド設定ファイル（`gv3.keys.toml`）
- 詳細は `docs/keybindings.md` を参照
