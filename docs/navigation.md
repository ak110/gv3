# ナビゲーション

キーボードとマウスによる画像間の移動操作です。

## 主要キーバインド

| キー              | 操作                      |
| ----------------- | ------------------------- |
| ← / →             | 前後の画像に移動          |
| ↑ / ↓             | 前後の画像に移動          |
| ホイール上/下     | 前後の画像に移動          |
| PageUp / PageDown | 5ページ移動               |
| Ctrl+PageUp/Down  | 50ページ移動              |
| Ctrl+Home / End   | 最初 / 最後へ             |
| Ctrl+ホイール     | 拡大 / 縮小               |
| Num /             | 自動縮小表示              |
| Num \*            | 自動縮小・拡大表示        |
| Num 0             | 余白トグル                |
| A                 | αチャネル背景切替         |
| Alt+Enter         | フルスクリーン            |
| Esc               | メニューバー表示/非表示   |
| F4                | ファイルリスト表示/非表示 |
| Shift+Space       | スライドショー開始/停止   |
| Shift+↑           | スライドショー加速        |
| Shift+↓           | スライドショー減速        |

全キーバインドの一覧は `ぐらびゅ.keys.default.toml` を参照してください。

## マウス操作

| 操作             | 動作                   |
| ---------------- | ---------------------- |
| 左ダブルクリック | ウィンドウ最大化トグル |
| ホイールクリック | 画像情報表示           |
| Ctrl+ホイール上  | 倍率を0.7倍する        |
| Ctrl+ホイール下  | 倍率を1.4倍する        |
| ホイール上/下    | 前へ/次へ（設定可能）  |
