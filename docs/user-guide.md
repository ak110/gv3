# ぐらびゅ ユーザーガイド

## 基本的な使い方

```cmd
REM 画像ファイルを指定して起動
ぐらびゅ.exe image.jpg

REM ファイル関連付けやD&Dでも起動可能
```

ドラッグ&ドロップでファイルまたはフォルダを開くこともできます。

## 画像の表示

### 表示モード

- **自動縮小表示**: ウィンドウサイズに収まるよう画像を縮小（デフォルト）
- **自動縮小・自動拡大表示**: ウィンドウサイズに合わせて拡大も行う
- **自動拡大表示**: ウィンドウサイズに合わせて拡大のみ行う
- **原寸大表示**: 100%表示
- 倍率の拡大(×1.4) / 縮小(×0.7) / リセット(×1.0)

### 余白・αチャネル

- 画像とウィンドウ端の余白量を設定可能（トグル）
- PNG等の透過画像に対して背景色を切り替え可能（白、黒、チェッカーパターン）

### 画像情報

- 画像サイズ（幅×高さ）、ファイルサイズ、ファイルパスをタイトルバーとメッセージボックスで表示

## ナビゲーション

画像ファイルを開くと、同一フォルダ内の画像ファイルを自動的にリストに追加します。
アーカイブファイルを開くと、内部の画像を一時フォルダに展開してリスト化します。

### 主要キーバインド

| キー | 操作 |
| --- | --- |
| ← / → | 前後の画像に移動 |
| ホイール上/下 | 前後の画像に移動 |
| PageUp / PageDown | 5ページ移動 |
| Ctrl+PageUp/Down | 50ページ移動 |
| Ctrl+Home / End | 最初 / 最後へ |
| Ctrl+ホイール | 拡大 / 縮小 |
| Num / | 自動縮小表示 |
| Num * | 自動縮小・拡大表示 |
| Num 0 | 余白トグル |
| A | αチャネル背景切替 |
| Alt+Enter | フルスクリーン |
| Esc | メニューバー表示/非表示 |
| F4 | ファイルリスト表示/非表示 |

全キーバインドの一覧は `ぐらびゅ.keys.default.toml` を参照してください。

## ファイル操作

### マーク機能

ファイルにマークを設定し、マークしたファイルに対して一括操作（削除、移動、複製）ができます。
マーク間のナビゲーションも可能です。

### 単一ファイル操作

- ファイルをリストから削除（ファイル自体は残る）
- ファイルの削除（ごみ箱経由）、移動、複製
- ファイル名・画像のクリップボードコピー
- クリップボードからの貼り付け
- JPG/BMP/PNGへの書き出し

## 画像編集

### 矩形選択

マウスドラッグで矩形領域を選択できます。
選択範囲は8方向リサイズハンドルでサイズ変更、ドラッグ移動が可能です。

### トリミング・回転・リサイズ

- 選択範囲でのトリミング
- 水平/垂直反転、90°/180°/任意角度回転（バイリニア補間）
- Lanczos3リサンプリングによるリサイズ

### フィルタ

パラメータあり: 塗り潰し、レベル補正、ガンマ補正、明るさとコントラスト、モザイク、ガウスぼかし、アンシャープマスク

パラメータなし: 色の反転、グレースケール（簡易/厳密）、α合成、ぼかし（通常/強い）、シャープ（通常/強い）、メディアンフィルタ

すべてのフィルタは選択範囲がある場合、その領域にのみ適用されます。

### 永続フィルタ

画像のナビゲーション（切り替え）を行っても全画像に自動的に適用されるフィルタです。
有効/無効のトグル、フィルタ操作の追加（順序通りに適用）、全操作のクリアができます。

## ファイルリスト管理

### ソート

ファイル名順（デフォルト）、大文字小文字区別なし、ファイルサイズ順、最終更新日時順、数値認識ソート（自然順ソート）に対応しています。

### シャッフル

ファイルリスト全体のシャッフル、フォルダの並び順のシャッフルができます。

### ブックマーク

現在のファイルリストと表示位置を独自テキスト形式（`.gvbm`）で保存・復元できます。

## 対応フォーマット

### 画像（標準対応）

JPEG, PNG, GIF, BMP, WebP

### PDF

Windows.Data.Pdf APIによるPDFレンダリングに対応しています。
各ページを画像として表示し、フォルダ内と同様にページ間を移動できます。

### アーカイブ

ZIP / cbz, RAR / cbr, 7z

### Susieプラグイン

64bit Susieプラグイン (.sph / .spi) に対応しています。
実行ファイルと同じディレクトリの `spi/` フォルダにプラグインDLLを配置すると自動検出されます。

```text
ぐらびゅ.exe
spi/
  ifXXX.sph    ← 画像プラグイン
  axXXX.spi    ← アーカイブプラグイン
```

## シェル統合

```cmd
REM ファイル関連付け・コンテキストメニュー・「送る」を一括登録
ぐらびゅ.exe --register

REM 一括解除
ぐらびゅ.exe --unregister
```

## 設定のカスタマイズ

`ぐらびゅ.toml` をexeと同じディレクトリに配置することで設定をカスタマイズできます。
同梱の `ぐらびゅ.default.toml` をコピーしてリネームしてお使いください。

## キーバインドのカスタマイズ

`ぐらびゅ.keys.toml` をexeと同じディレクトリに配置することで、ほぼ全ての操作のキーバインドをカスタマイズできます。
同梱の `ぐらびゅ.keys.default.toml` をコピーしてリネームし、編集してください。

### TOML構造の例

```toml
# ぐらびゅ.keys.toml
# 複数キーの割り当て: カンマ区切り（例: "Ctrl+Num -, Ctrl+WheelUp"）

[file]
new_window = "Ctrl+N"
open_file = "Ctrl+O"
open_folder = "Ctrl+Shift+O"
close_all = "Ctrl+W"

[mark]
set = "Delete"
unset = "Ctrl+Delete"
invert_all = "Ctrl+Shift+I"
# ... 以下同様
```

### キー名の記法

| 記法 | キー |
| --- | --- |
| `←` `→` `↑` `↓` | 矢印キー |
| `PageUp` `PageDown` | ページアップ/ダウン |
| `Home` `End` | Home / End |
| `Enter` `Space` `Tab` `Esc` `BackSpace` | 各種キー |
| `Delete` | Delete |
| `F1` 〜 `F12` | ファンクションキー |
| `Num0` 〜 `Num9` | テンキー数字 |
| `Num +` `Num -` `Num *` `Num /` | テンキー演算子 |
| `Ctrl+` `Shift+` `Alt+` | 修飾キー（組み合わせ可） |

## マウス操作

| 操作 | 動作 |
| --- | --- |
| 左ダブルクリック | ウィンドウ最大化トグル |
| ホイールクリック | 画像情報表示 |
| Ctrl+ホイール上 | 倍率を0.7倍する |
| Ctrl+ホイール下 | 倍率を1.4倍する |
| ホイール上/下 | 前へ/次へ（設定可能） |

## ピクセルグリッド表示

高倍率ズーム時にピクセル境界をグリッド線で表示します。
