# はじめに

## インストール

[Releases](https://github.com/ak110/gv/releases)からZIPをダウンロードし、任意のフォルダに展開する。

## 基本的な使い方

ファイル関連付けやドラッグ&ドロップで画像ファイル・フォルダを開ける。
コマンドラインから `ぐらびゅ.exe image.jpg` のように指定して起動できる。

画像ファイルを開くと、同一フォルダ直下の画像ファイルを自動的にリストに追加する（サブフォルダは対象外）。
フォルダを開くと、サブフォルダ内の画像・PDF・アーカイブまで再帰的に取り込み、親フォルダから順にサブフォルダの内容を続けて並べる。
アーカイブファイルを開くと、内部の画像を一時フォルダに展開してリスト化する。

## コマンドライン引数

| 引数           | 動作                                                       |
| -------------- | ---------------------------------------------------------- |
| ファイルパス   | 指定した画像ファイルやフォルダを開く                       |
| `--register`   | ファイル関連付け・コンテキストメニュー・「送る」を一括登録 |
| `--unregister` | 一括解除                                                   |
