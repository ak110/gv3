# 画像編集

矩形選択、トリミング、回転、フィルタなどの画像編集機能である。

## 矩形選択

マウスドラッグで矩形領域を選択できる。
選択範囲は8方向リサイズハンドルでサイズ変更、ドラッグ移動が可能である。

## トリミング・回転・リサイズ

- 選択範囲でのトリミング
- 水平/垂直反転、90°/180°/任意角度回転（バイリニア補間）
- Lanczos3リサンプリングによるリサイズ

## フィルタ

選択範囲がある場合は、その領域にのみフィルタを適用する。

- 色調補正: レベル補正、ガンマ補正、明るさとコントラスト、色の反転、グレースケール（簡易/厳密）
- ぼかし・シャープ: ガウスぼかし、ぼかし（通常/強い）、アンシャープマスク、シャープ（通常/強い）、メディアンフィルタ
- 効果: 塗り潰し、モザイク、α合成

## 永続フィルタ

画像のナビゲーション（切り替え）を行っても全画像に自動的に適用されるフィルタである。
有効/無効のトグル、フィルタ操作の追加（順序通りに適用）、全操作のクリアができる。
