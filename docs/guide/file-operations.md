# ファイル操作

マーク、削除、移動、複製などのファイル操作機能である。

## マーク機能

ファイルにマークを設定し、マークしたファイルに対して一括操作（削除、移動、複製）ができる。
マーク間のナビゲーションも可能である。

## 単一ファイル操作

- ファイルをリストから削除（ファイル自体は残る）
- ファイルの削除（ごみ箱経由）、移動、複製
- ファイル名・画像のクリップボードコピー
- クリップボードからの貼り付け
- JPG/BMP/PNGへの書き出し

## ファイルリスト管理

### ソート

ファイル名順（デフォルト）、大文字・小文字の区別なし、ファイルサイズ順、最終更新の日時順、
数値認識ソート（自然順ソート）に対応している。

### シャッフル

ファイルリスト全体のシャッフル、フォルダの並び順のシャッフルができる。

### ブックマーク

現在のファイルリストと表示位置を独自テキスト形式 (`.gvbm`) で保存・復元できる。
