# 画像の表示

画像の表示モード、ズーム、スライドショーなどの表示機能について述べる。

## 表示モード

- 自動縮小表示: ウィンドウサイズに収まるよう画像を縮小（デフォルト）
- 自動縮小・自動拡大表示: ウィンドウサイズに合わせて拡大も行う
- 自動拡大表示: ウィンドウサイズに合わせて拡大のみ行う
- 原寸大表示: 100%表示
- 倍率の拡大（×1.4）/ 縮小（×0.7）/ リセット (×1.0)

## 余白・αチャネル

- 画像とウィンドウ端の余白量を設定可能（トグル）
- PNG等の透過画像に対して背景色を切り替え可能（白、黒、チェッカーパターン）

## 画像情報

- 画像サイズ（幅×高さ）、ファイルサイズ、ファイルパスをタイトルバーとメッセージボックスで表示
- JPEG等のEXIF情報（カメラ、レンズ、焦点距離、F値、シャッター速度、ISO感度、撮影日時、GPS座標）を画像情報ダイアログに表示

## スライドショー

Spaceキーで自動送りを開始/停止できる。Shift+↑で加速、Shift+↓で減速する（500ms〜30秒）。
最後の画像に到達すると最初に戻る（設定で無効化可能）。
手動でナビゲーションを行うと自動的に停止する。フルスクリーンモードと併用できる。

デフォルトの間隔やリピート設定は[カスタマイズ](./customization.md)で説明する設定ファイルの`[slideshow]`セクションで変更できる。

## ピクセルグリッド表示

高倍率ズーム時にピクセル境界をグリッド線で表示する。
