# ぐらびゅ3 キーバインド仕様

## カスタマイズ方式

キーバインドは `gv3.keys.toml` ファイルで定義する。
ユーザーはこのファイルを編集して全てのキーバインドをカスタマイズ可能。

```toml
# gv3.keys.toml
# キー名: "Ctrl+O", "Shift+Delete", "Alt+Enter", "F5", "←", "→" 等
# 複数キーの割り当て: カンマ区切り（例: "Ctrl+Num -, Ctrl+WheelUp"）

[file]
new_window = "Ctrl+N"
open_file = "Ctrl+O"
open_folder = "Ctrl+Shift+O"
open_history = "Ctrl+H"
close_all = "Ctrl+W"

[mark]
set = "Delete"
unset = "Ctrl+Delete"
invert_all = "Ctrl+Shift+I"
# ... 以下同様
```

## マウス操作

| 操作 | 動作 |
|------|------|
| 左ダブルクリック | ウィンドウ最大化トグル |
| 右クリック | （将来: コンテキストメニュー表示） |
| ホイールクリック | 画像情報表示 |
| Ctrl+ホイール上 | 倍率を0.7倍する |
| Ctrl+ホイール下 | 倍率を1.4倍する |
| ホイール上/下 | 前へ/次へ（設定可能） |

## キー名の記法

設定ファイルで使用するキー名:

| 記法 | キー |
|------|------|
| `←` `→` `↑` `↓` | 矢印キー |
| `PageUp` `PageDown` | ページアップ/ダウン |
| `Home` `End` | Home / End |
| `Enter` `Space` `Tab` `Esc` `BackSpace` | 各種キー |
| `Delete` | Delete |
| `F1` 〜 `F12` | ファンクションキー |
| `Num0` 〜 `Num9` | テンキー数字 |
| `Num +` `Num -` `Num *` `Num /` | テンキー演算子 |
| `Ctrl+` `Shift+` `Alt+` | 修飾キー（組み合わせ可） |
