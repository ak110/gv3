# 画像編集

## 矩形選択

マウスドラッグで矩形領域を選択できます。
選択範囲は8方向リサイズハンドルでサイズ変更、ドラッグ移動が可能です。

## トリミング・回転・リサイズ

- 選択範囲でのトリミング
- 水平/垂直反転、90°/180°/任意角度回転（バイリニア補間）
- Lanczos3リサンプリングによるリサイズ

## フィルタ

パラメータあり: 塗り潰し、レベル補正、ガンマ補正、明るさとコントラスト、モザイク、ガウスぼかし、アンシャープマスク。

パラメータなし: 色の反転、グレースケール（簡易/厳密）、α合成、ぼかし（通常/強い）、シャープ（通常/強い）、メディアンフィルタ。

すべてのフィルタは選択範囲がある場合、その領域にのみ適用されます。

## 永続フィルタ

画像のナビゲーション（切り替え）を行っても全画像に自動的に適用されるフィルタです。
有効/無効のトグル、フィルタ操作の追加（順序通りに適用）、全操作のクリアができます。
